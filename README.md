## installations

Install Java

```
sudo apt-get install openjdk-11-jdk
```

Install maven

```
sudo apt install maven
```

## Usage

Compile using 

```
mvn compile
```

Create a jar using 

```
mvn package
```

Run main using 

```
java -cp target/Bitwise_Calculator-1.0-SNAPSHOT.jar Calculator
```
